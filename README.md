# httplite
httplite is a C++ class library for doing HTTP client and server programming with no dependencies. \
The httplite directory contains the class library you would link against for HTTP functionality. \
The HttpClient and HttpServer classes are what you integrate with.

## httpclient
The httpclient directory contains a proof of concept command line HTTP client application.

## httpserver
The httpserver directory contains a proof of concept command line HTTP server application.

## tests
The tests directory contains unit tests for all the classes in httplite.
